Os scripts contidos nessa pasta empacotam o .jar em um executável .exe

# Usos
## criar runtime image
Coloque o .jar na pasta input
Edite o script preenchendo as informações do aplicativo
Ao executar o script, uma runtime image do app será criado na pasta output

## isa-setup
Abra o arquivo usando o Inno Setup
Edite as configurações de acordo com as informações do seu aplicativo
Ao compilar, um instalador será gerado na pasta output
